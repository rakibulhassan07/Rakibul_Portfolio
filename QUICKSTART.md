# Quick Start Guide

## Run the Development Server

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to see your hero section in action!

## What You Get

✅ Full-screen hero section with:
- Dark gradient background (gray → purple → gray)
- Bold gradient text headings
- Smooth fade-in + slide-up animations  
- Responsive typography
- "View My Work" CTA button with hover effects
- Animated scroll indicator

## Customize the Content

Edit `components/HeroSection.tsx` to change:
- Your name/title text
- Subtitle description
- Button text and action
- Colors and gradients
- Animation timing

## Next Steps

1. **Add more sections** below the hero (About, Projects, Contact)
2. **Customize colors** to match your brand
3. **Add a work section** for the button to scroll to
4. **Deploy** to Vercel, Netlify, or your hosting provider

Enjoy your new portfolio! 🚀
